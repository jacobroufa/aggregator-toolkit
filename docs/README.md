# Documenters Municipality Toolkit

The Municipality Toolkit is a front-end for the [Documenters Aggregator](/City-Bureau/documenters-aggregator), and is intended for use by municipalities in order to facilitate the process of scraping and recording meeting information.

* [Installation](installation.md)
